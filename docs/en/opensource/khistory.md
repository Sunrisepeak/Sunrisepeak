## 1. Features

- Keyboard key detection
- Gamepad key detection
- Control
  - Key history count
  - Transparency
  - Detection frame rate
  - Visualization plugin selection
- Key visualization and extension (custom key layout/key remapping/custom key highlight colors/others)
